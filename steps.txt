‚≠ê Step 2: Extract Roads, Buildings, POIs Separately

OSM raw contains everything mixed, so split it:

Tool 1: geojson.io

Open your geojson

Select only buildings

Export as buildings.geojson

Repeat for roads, paths, areas

Tool 2 (better): osmnx (Python)

You can run:

import osmnx as ox

g = ox.graph_from_xml("map.osm")
ox.save_graphml(g, "graph.graphml")


This gives you:
‚úî road network
‚úî walkable graph
‚úî ready for routing

‚≠ê Step 3: Build the routing engine

Choose one:

Option A (easy): GraphHopper

Download GraphHopper

Run:

graphhopper import map.osm


It generates a routing graph you can use in your app.

Option B (faster): OSRM
osrm-extract map.osm -p profiles/car.lua
osrm-contract map.osm.osrm


This gives:

turn-by-turn instructions

geometry for route

waypoints
